EXP_01 TEST FIR: KEVLAR vs NYLON ELASTICITATE
// DESCRIERE EXPERIMENT
Două fire sunt suspendate și li se aplică greutăți identice. Diferența de comportament relevă proprietățile mecanice fundamentale ale fiecărui material.

Stânga — Fir Kevlar: Firul abia se mișcă și nu se curbează. Kevlarul prezintă o stabilitate dimensională excelentă și o rigiditate extremă — nu se alungește practic deloc sub tensiune.

Dreapta — Fir Nylon: Firul se întinde și se lasă semnificativ în jos. Nailonul este un material foarte elastic, care se alungește considerabil sub aceeași sarcină statică.

Greutățile aplicate sunt identice — diferența de deformare provine exclusiv din proprietățile moleculare ale celor două materiale.

EXP_02 TEST TĂIERE TĂIERE
// DESCRIERE EXPERIMENT
Se încearcă tăierea Kevlarului cu un cutter obișnuit cu lama nouă. Forța aplicată este semnificativă, dar fibra rezistă tăierii directe.

Structura țesută a fibrei distribuie forța de-a lungul mai multor fire, împiedicând propagarea tăieturii. De aceea Kevlarul este folosit în mănuși anti-tăiere și armuri.

EXP_03 ARDERE KEVLAR TERMICĂ
// DESCRIERE EXPERIMENT
Kevlarul este expus direct la flacără. Observăm că fibra nu se topește și nu se inflamează — carbonizează local, menținând integritatea structurală a zonelor neafectate.

Punctul de descompunere al Kevlarului este de ~450°C, față de polietilenă (~120°C). Această proprietate îl face esențial în echipamente de pompieri și blindaje termice.

EXP_04 ARDERE NYLON TERMICĂ
// DESCRIERE EXPERIMENT — COMPARAȚIE
Nylonul (poliamidă alifatică) este expus la aceeași flacără. Spre deosebire de Kevlar, nylonul se topește și se contractă rapid, formând picături arzânde.

Punctul de topire al nylonului este ~220°C — de două ori mai mic decât temperatura de descompunere a Kevlarului. Comportamentul este complet diferit, confirmând superioritatea structurii aromatice din Kevlar.

EXP_05 ARDERE BUMBAC TERMICĂ
// DESCRIERE EXPERIMENT — COMPARAȚIE
Bumbacul arde continuu și rapid, fără autoextingere, producând flacără vizibilă și cenușă. Celuloza din bumbac este extrem de inflamabilă.

Comparând cele trei materiale, Kevlarul demonstrează o superioritate termică clară: carbonizare controlată vs. topire (Nylon) vs. ardere completă (Bumbac).
